01 · ผลิตภัณฑ์

PBAT คืออะไร และเหนือกว่าพลาสติกทั่วไปอย่างไร

พลาสติกย่อยสลายได้ทางชีวภาพที่ผ่านมาตรฐาน Compostable จริง ใช้ทดแทน PE/LDPE ได้ — ยืดหยุ่น ทนทาน สลายเป็นดินและน้ำโดยไม่เหลือสารไมโครพลาสติก

🧬

วัสดุ PBAT

Polybutylene Adipate Terephthalate (BDO + AA + PTA) — ต้นทุนเม็ดพลาสติกลดลงมากจากกระบวนการผลิตถ่านหินของจีน ทำให้ราคาสามารถแข่งขันกับ PE/PP ได้

♻️

ย่อยสลายได้จริง

Home Compost ~12 เดือน · Industrial Compost ~6 เดือน กลายเป็นดิน+น้ำ ไม่เหลือสารไมโครพลาสติก ตามมาตรฐาน EN 13432 / ISO 17088

🛡️

ผ่านการทดสอบเชิงกล

Drop Test · Jog Test (ทนแรงดึง 1,000 ครั้ง) · Static Load — ความแข็งแรงเพียงพอต่อการใช้งานจริงในระดับการค้าปลีกและ HORECA

หมายเหตุ: การทดสอบเชิงกลแตกต่างจากการรับรองมาตรฐาน Compostable — ดูสถานะใบรับรองในส่วน ESG

ดีมานด์เพิ่มขึ้นจากกระแส ESG และกฎหมาย ขณะที่ต้นทุนลดลง — ผู้ที่ชนะตลาดคือผู้ที่มีกำลังผลิตพร้อม ระบบเอกสารครบ และใบรับรองมาตรฐาน

ความสำเร็จในตลาดนี้ไม่ได้ขึ้นอยู่กับทักษะการขายเพียงอย่างเดียว
แต่ขึ้นอยู่กับ ความพร้อมด้านเอกสาร · ความสามารถในการส่งมอบจริง · และความน่าเชื่อถือของใบรับรองมาตรฐาน

03 · ภูมิทัศน์การแข่งขัน

เปรียบเทียบคู่แข่ง — และช่องที่ CPPC เข้าแทรกได้

ผู้แข่งขันแต่ละรายมีจุดอ่อนเฉพาะตัว ตารางนี้ชี้ "ช่องว่างตลาด" ที่ CPPC สามารถเข้าแทรกด้วยข้อได้เปรียบที่ผู้อื่นไม่มีพร้อมกัน

ผู้เล่นในตลาด จุดเด่น จุดด้อย / ข้อจำกัด ช่องให้ CPPC เข้าแทรก
TPBIไทย · บจ. มหาชน · แบรนด์ "BioPlus" แบรนด์ระดับประเทศ เน้นส่งออก · ได้รับใบรับรอง EN 13432 (DIN CERTCO) เน้นตลาดส่งออกและราคาพรีเมียม · ไม่มีช่องทางในเครือ CP ใช้ช่องทาง CP ALL / CP AXTRA (Makro-Lotus) + ความได้เปรียบด้านต้นทุนเรซินในกลุ่ม CP
Thantawan Industryไทย · ประสบการณ์ 40+ ปี ใบรับรองครบ (Seedling EU + BPI) · ความน่าเชื่อถือสูงในวงการ กำลังผลิตและช่องทางจำหน่ายจำกัด · ไม่ใช่ผู้นำด้านต้นทุน กำลังผลิตเป้า 2,000 ตัน/ด. + ทีมขายภาคสนามทุกภูมิภาค + Supply Continuity
Bio Sure / Bags&Glovesไทย · SME ผลิตจาก PBAT/PLA จริง · มีฉลาก TÜV / GC กำลังผลิตจำกัด · ช่องทางการจำหน่ายแบบ "ขอใบเสนอราคา" — ไม่มีบริการภาคสนาม ความต่อเนื่องของการส่งมอบ + บริการและการสนับสนุนทั่วประเทศ
สินค้านำเข้าจากจีนKingfa · Wanhua · OEM ต่างๆ ราคาต่ำสุดในตลาด · กำลังผลิตล้นเกิน (~15 ล้านตัน/ปี) Lead Time ยาว · ไม่มีการบริการหลังการขายในไทย · ขาดใบรับรองตามมาตรฐานไทย · ความเสี่ยงด้านคุณภาพ ผลิตในประเทศ + บริการหลังการขาย + ใบรับรองมาตรฐาน + ส่งมอบรวดเร็ว
CPPC (บริษัทของเรา)ไทย · โรงงานในเครือ CP · สีคิ้ว กำลังผลิตระดับโรงงาน + ช่องทางในเครือ CP + ต้นทุนเรซินกลุ่ม CP + ทีมขายภาคสนามทุกภาค ส่วนต่างราคากับ PE ยังสูง (แก้ด้วยกลยุทธ์ Gauge Optimisation) · ใบรับรอง Compostable อยู่ระหว่างดำเนินการ (คาด ก.ค.–ก.ย. 2569) จุดยืนที่ชัดเจน: รวมทุกข้อได้เปรียบที่คู่แข่งไม่มีพร้อมกันในรายเดียว

ช่องว่างตลาด = โอกาสเชิงกลยุทธ์ของ CPPC

ไม่มีผู้แข่งขันรายใดในตลาดที่ รวมกำลังผลิตระดับโรงงาน + ช่องทางในเครือ CP + ทีมขายภาคสนามทุกภูมิภาค + ต้นทุนเรซินจากกลุ่ม CP ได้พร้อมกันในรายเดียว — นั่นคือจุดยืนที่แตกต่างและแข่งขันได้อย่างยั่งยืน

เครือ CP = ช่องทาง + ต้นทุนเรซิน เป้ากำลังผลิต 2,000 ตัน/ด. Supply Continuity Field Sales ทุกภูมิภาค Gauge Optimisation

วัตถุดิบทางเลือกในประเทศ: PTT MCC BioPBS (GC×Mitsubishi, 20,000 ตัน/ปี) · Thai Wah TPS — ส่วนแบ่งตลาดรายบริษัทยังขาดข้อมูลสาธารณะ Pending Validation ควรดึงข้อมูลจากกรมพัฒนาธุรกิจการค้า (DBD)

05 · ราคาและกลยุทธ์ส่วนต่าง

โครงสร้างราคา & กลยุทธ์ลดส่วนต่าง

อุปสรรคหลักคือราคา ไม่ใช่ความต้องการ — ลูกค้าประเมินจาก "ต้นทุนต่อการใช้งาน + การลดความเสี่ยงด้านแบรนด์" ไม่ใช่แค่บาท/กิโลกรัม

ระดับส่วนต่างราคาที่ลูกค้ายอมรับ (เทียบกับ PE) Internal Estimate

0–5% — ขายได้ทันที (สเปคใกล้เคียง + ใบรับรองครบ)ง่าย
5–12% — ขายได้เมื่อมี ESG KPI หรือแคมเปญปานกลาง
>12% — ขายยาก เฉพาะโครงการ Pilot หรือ Premium Brandยาก

สถานการณ์ปัจจุบัน CPPC: ถุงหูหิ้ว +14–16% · ถุงกระตุก +6–26% → จำเป็นต้องใช้กลยุทธ์ Gauge Optimisation เพื่อดึงราคาเข้าโซน ≤12%

กลยุทธ์การปรับความหนา (Gauge Optimisation)

+25% → +7%

ถุงกระตุก: ลดความหนา 15→10 ไมครอน/ด้าน ตัดส่วนต่างราคาจาก ~25% เหลือ ~7% โดยยังใช้งานได้ตามปกติ

ข้อมูลตลาดออนไลน์เพื่อการวิเคราะห์

  • ผู้ขายส่วนใหญ่ ไม่แสดงราคา List — ใช้ระบบ "ขอใบเสนอราคา" ยืนยันว่าตลาดนี้เป็น B2B เชิงความสัมพันธ์
  • ราคาขายส่งถุงขยะ Compostable ~85–90 ฿/กก. (MOQ 1,000 กก.) Internal Estimate
  • ตลาด Retail: แบรนด์ "Smarter" วางจำหน่ายถุงขยะย่อยสลายบน Tops Online
  • ข้อสรุป: การแข่งขันจริงอยู่ใน Offline Relationship Selling — ซึ่งเป็นจุดแข็งของ CPPC

07 · แนวทางการขาย (Field Sales)

แนวทางสร้างความได้เปรียบในการแข่งขันของ CPPC

จุดแข็ง CPPC คือโรงงานผลิตจริง + ทีมขายทุกภูมิภาค — ดำเนินการขายแบบ Account-Based Selling พร้อมส่งตัวอย่างและทดสอบร่วมกับลูกค้า ซึ่งผู้นำเข้าและผู้ขายออนไลน์ทำแทนไม่ได้

1

ชนะด้านข้อกำหนดทางเทคนิค (Win the Spec)

ส่ง Sample Kit + เอกสารทางเทคนิค (ความหนา, แรงดึง, Static Load, การปิดผนึก, Drop Test, แผนการรับรองมาตรฐาน) ให้ฝ่าย QA/Packaging อนุมัติก่อน — การตัดสินใจของฝ่ายจัดซื้อขึ้นอยู่กับความพร้อมด้านเอกสารและการรับรองมาตรฐาน

2

ชนะด้านการทดลองใช้ (Win the Pilot)

คัดเลือก 6–10 บัญชีนอกเครือ กำหนด KPI (ข้อร้องเรียน/ล้านใบ, ความเสียหาย, OTIF) → เมื่อ Pilot ผ่าน เปลี่ยนเป็น Framework Deal ทันที

3

ชนะด้านกระบวนการจัดซื้อ (Win the Procurement)

เตรียม 3 ชุดเอกสาร: Commercial (ราคา/สูตรปรับตามต้นทุน) · Technical (COA/Spec/Test Report) · Sustainability (ใบรับรอง/Claim/Logo Guideline)

📑

โครงสร้างสัญญา 3 ระดับ

Pilot PO 1–3 เดือน → Annual Framework + Monthly Release (ผูกสูตรปรับตาม Resin/FX) → Co-Development (Private Label)

🎯

ใบรับรอง Home Compost จริง · ไม่เหลือสารไมโครพลาสติก · Carbon Footprint ต่อใบที่วัดได้ · ความต่อเนื่องในการส่งมอบ · กลยุทธ์ Gauge Optimisation

08 · สรุปเชิงกลยุทธ์

มัด Anchor Accounts ในเครือและนอกเครือ

ยึดลูกค้าในเครือ (CP ALL / CP AXTRA) + คัดเลือก 6–10 รายนอกเครือที่มีมาร์จินสูงในช่วงที่กำลังผลิตยังจำกัดในปี 2569 — ใช้ทีมขายภาคสนามบุกตามลำดับพื้นที่

จัดทำ Spec × Price Matrix + กลยุทธ์ Gauge Optimisation

จัดทำเครื่องมือสำหรับทีมขายในการนำเสนอและปิดดีล ดึงส่วนต่างราคาเข้าโซน ≤12% ที่ลูกค้าสามารถยอมรับได้ โดยใช้กลยุทธ์การปรับความหนาที่เหมาะสมกับการใช้งานของลูกค้าแต่ละราย
