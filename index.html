01 · ผลิตภัณฑ์

PBAT คืออะไร และดีกว่าตรงไหน

พลาสติกย่อยสลายได้ทางชีวภาพ (compostable จริง) ใช้แทน PE/LDPE — ยืดหยุ่น ทนทาน ย่อยเป็นดินและน้ำ ไม่เหลือไมโครพลาสติก

🧬

วัสดุ PBAT

โพลีบิวทิลีนอะดิเพตเทเรฟทาเลต (BDO + AA + PTA) เม็ดต้นทุนลดลงมากจากรูทถ่านหินจีน ทำให้ราคาเทียบเท่า PE/PP ได้

♻️

ย่อยสลายจริง

Home compost ~12 เดือน · Industrial compost ~6 เดือน กลายเป็นดิน+น้ำ ไม่เป็นไมโครพลาสติก

🛡️

ผ่านการทดสอบ

Drop test · Jog test (หูหิ้วถึง 1,000 ครั้ง) · Static load — แข็งแรงเพียงพอต่อการใช้งานจริง

เทียบสเปคกับถุง PE

ที่สเปคเทียบเท่ากัน

น้ำหนัก PBAT มากกว่า PE 20–30% คือ "ตัวขับต้นทุน" เพราะขายเป็นบาท/กก. → คันโยกคือการลดความหนา (gauge)

05 · ราคา

ราคา & กำแพง Premium

กำแพงหลักคือราคา ไม่ใช่ดีมานด์ — ลูกค้าตี "ต้นทุนต่อการใช้ + ลดความเสี่ยงแบรนด์" ไม่ใช่บาท/กก. อย่างเดียว

ลูกค้ายอมจ่ายแพงขึ้นเท่าไร (vs PE) 🟡

0–5% · ขายง่ายสุด (สเปคใกล้เดิม+เซอร์ครบ)ง่าย
5–12% · ได้เมื่อมี ESG KPI / แคมเปญปานกลาง
>12% · ขายยาก (เฉพาะ premium/pilot)ยาก

CPPC ตอนนี้: หูหิ้ว +14–16% · กระตุก +6–26% → ต้องใช้เกม gauge ดึงเข้าโซน ≤12%

🔧 เกม Gauge = คันโยกปิดราคา

+25% → +7%

ถุงกระตุกลดความหนา 15→10 mic/side ตัด premium จาก ~25% เหลือ ~7% โดยยังใช้งานได้

💻 ตลาดออนไลน์เขาขายกันยังไง 🟢

  • ส่วนใหญ่ ไม่โชว์ราคา list — ขายแบบ "ขอใบเสนอราคา" = ตลาด B2B เชิงสัมพันธ์
  • ขายส่งถุงขยะ compostable ~85–90 ฿/กก. (MOQ 1,000 กก.) 🟡
  • Retail: แบรนด์ "Smarter" ถุงขยะย่อยสลายวางขายบน Tops Online
  • แปลว่า: เกมจริงคือ offline relationship selling → เข้าทาง CPPC

07 · กลยุทธ์การขาย (Offline Field Sales)

ขายให้ต่าง เพื่อปิดเป้าที่วางไว้

จุดแข็ง CPPC คือโรงงานผลิต + ทีมขายทุกภูมิภาค — ทำ account-based selling, ส่งตัวอย่าง, ทดสอบร่วม แบบ face-to-face ที่คู่แข่งนำเข้า/ออนไลน์ทำไม่ได้

1

Win the Spec

ส่ง sample + sales kit (thickness, tensile, static load, seal, drop, cert roadmap) ให้ QA/Packaging ผ่านก่อน — buyer แพ้ที่เอกสารครบ

2

Win the Pilot

เลือก 6–10 account นอกเครือ ตั้ง KPI (complaint/ล้านใบ, breakage, OTIF) → ชนะแล้วรีบเปลี่ยนเป็น framework deal

3

Win the Procurement Story

เตรียม 3 ชุดเอกสาร: Commercial (ราคา/สูตรปรับ) · Technical (COA/spec/test) · Sustainability (cert/claim/logo)

📑

Contract 3 ขั้น

Pilot PO 1–3 ด. → Annual framework + monthly release (ผูกสูตร resin/FX) → Co-development (private label)

🎯

จุดต่างที่ขายได้

home-compost certified · ไม่เป็นไมโครพลาสติก · carbon ต่อใบ · supply continuity · เกม gauge

📦

ปี 2026 = เกมจัดสรร

ของยังไม่พอ (หูหิ้ว 832/3,678 ตัน) → lock ลูกค้ามาร์จินดี + win-the-spec ไว้ก่อน เร่ง ramp 2027

💬 Core Message

"ถุงย่อยสลายได้ที่มีใบรับรองจริง พร้อมซัพพลายระดับโรงงาน สำหรับค้าปลีก/ร้านอาหารที่ต้องการเปลี่ยนแบบไม่เสี่ยง"
