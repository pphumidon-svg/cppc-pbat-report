# โปรเจค: AI Agent ที่เรียนรู้จาก E-book

## บริบทโปรเจค

เจ้าของโปรเจคต้องการพัฒนา AI Agent ที่ **มีตัวตนและเติบโตได้** ผ่านการอ่าน e-book
ไม่ใช่แค่ระบบ RAG หรือ knowledge base ธรรมดา แต่คือ Agent ที่:
- พัฒนา **อุปนิสัย** และ **วิธีพูด** เฉพาะตัวจากสิ่งที่อ่าน
- มีกระบวนการ **ตกตะกอน** ความรู้ (ไม่ใช่แค่จำ)
- เข้าใจความเป็นมนุษย์ในเชิงลึก — อารมณ์ ภาษา ความนึกคิด

## โครงสร้างระบบ

```
ai_brain/
├── brain.py          → GrowingBrain class — หัวใจของระบบ
├── chat.py           → คุยกับ Agent แบบ interactive
├── demo.py           → ทดสอบระบบ
├── brain_data/
│   └── brain.json    → ตัวตนของ Agent (เติบโตทุกครั้งที่อ่านหนังสือ)
└── START_HERE.md     → วิธี setup บนเครื่อง local
```

## กระบวนการเรียนรู้ (4 ขั้น)

1. `read_with_soul()` — อ่านแบบมีความรู้สึก ตั้งคำถาม เชื่อมโยง
2. `inner_dialogue()` — ถกเถียงกับตัวเองเพื่อให้ลึกขึ้น
3. `integrate_into_self()` — บรรจุลง brain.json อัปเดตตัวตน
4. `speak_as_myself()` — ตอบในแบบที่ตัวเองเป็น ไม่ใช่ generic AI
